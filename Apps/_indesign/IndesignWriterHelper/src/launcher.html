üöÄ Quick Start

If you're having issues with the InDesign Writer Helper, follow these steps to get it working:

1. Check Python Installation

Make sure Python is installed on your system:

python --version

If Python is not found, download and install it from python.org or the Microsoft Store.

2. Install Required Dependencies

Install the required Python packages:

pip install flask pywin32 requests jinja2 werkzeug

3. Verify InDesign Installation

Make sure Adobe InDesign is installed on your system. The application needs InDesign to be installed to function properly.

4. Run as Administrator (if needed)

If you encounter permission issues, try running the batch file as administrator.

5. Check Firewall Settings

Make sure your firewall allows the application to access the network (port 8081).

üö® Common Issues & Solutions

‚ùå "Python not found"

Solution: Install Python from python.org and make sure to check "Add Python to PATH" during installation.

‚ùå "pywin32 module not found"

Solution: Run pip install pywin32 in Command Prompt. If it fails, try running as administrator.

‚ùå "InDesign not found"

Solution: Make sure Adobe InDesign is installed on your system. The application looks for InDesign in standard installation locations.

‚ùå "Port 8081 already in use"

Solution: Close other applications that might be using port 8081, or restart your computer.

‚ùå "Permission denied"

Solution: Run the batch file as administrator by right-clicking and selecting "Run as administrator".
